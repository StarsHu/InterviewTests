2. 题目
    ```python
    def ex(num, alist=[]):
        alist.append(num)
        return alist

    a = ex(10)
    b = ex(20, [])
    c = ex('a')
    print(a, b, c)
    ```
    答案
    ```python
    [10, 'a'] [20] [10, 'a']
    ```
